
# Agro-Smart
Proyecto multidiciplinario para gestión de sistema de riego automático para aportar a zonas de escasos recursos hídricos

# Instrucciones

Antes de iniciar los proyectos asegurense de tener instalados las versión de node 22.12.0 o posterior


# iniciar el proyecto backend
ingresar a la carpeta "backend" y escribir
npm i
node index.js


# Iniciar el proyecto Frontend
ingresar a la carpeta "frontend/app/" y escribir
npm i
npm run dev
